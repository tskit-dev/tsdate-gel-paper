# tsdate-gel-paper
Publicly shareable code and plot data for the "Tracing the evolutionary histories of ultra-rare variants using variational dating of large ancestral recombination graphs" manuscript

- The ``gel_analysis`` directory contains all details on the analyses performed on Genmomics England
  data. Please see the README in that directory for more details.
- The ``tgp_comparison`` directory contains a reproducible Snakemake workflow for generating the dataset
  of allele age estimates on 1000 Genomes data.
